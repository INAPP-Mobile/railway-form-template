Have a question or comment? Send us a message using the form below.

Installation Instructions

To embed this form in your Blogger blog:

  1. Deploy the backend — Click the "Deploy on Railway" button from the README and set the required environment variables (DATABASE_URL and ADMIN_PASSWORD).
  2. Get your app URL — After deployment, Railway gives your app a *.railway.app domain. Copy it.
  3. Replace the URL — In the form above, replace YOUR_APP.railway.app with your actual app URL.
  4. Add to Blogger:
    • Go to your Blogger dashboard → Layout → Add a Gadget
    • Choose "HTML/JavaScript"
    • Paste the form HTML (from the code block below)
    • Save and arrange the gadget where you want it
